Ultrasonic Range Finder Using Arduino

The URF is small, hand-sized Arduino device that measures the distance between itself and a still object using ultrasonic sound waves.
The device is powered with a Polymer Li-Ion Single Cell Battery 3.7V, 220mAh which allows for up to 9 hours of battery life.
This project taught me C++, Arduino programming and IoT development.
Skills used: C++ - Arduino - ARM Programming - Circuit Design
